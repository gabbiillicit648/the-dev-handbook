## 📚 What is the-dev-handbook?

the-dev-handbook is a free, open-source collection of courses designed to teach you how to code. It covers many popular programming languages and frameworks used today. This handbook helps you learn skills step by step, starting from the basics. You do not need any previous experience to get started.

7. The installation wizard will open. Follow the instructions on the screen:

8. After installation finishes, click "Finish" to close the installer.

9. Look for the shortcut icon named “the-dev-handbook” on your desktop or in your Start menu.

10. Double-click the icon to open the program.
